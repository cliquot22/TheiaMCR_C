# Security Policy

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub Issues.**

### Option 1 — GitHub Private Vulnerability Reporting (preferred)

Use GitHub's built-in private reporting tool:

1. Go to the **Security** tab of this repository.
2. Click **"Report a vulnerability"**.
3. Fill out the form describing the issue.

Your report will be visible only to repository maintainers until a fix is released.

### Option 2 — Email

If you prefer, you can report vulnerabilities by email:

**TheiaSecurity@theiatech.com**

Please include as much detail as possible: a description of the issue, steps to reproduce, affected versions, and any potential impact you've identified.

## Response

We aim to acknowledge reports within 1 week and will keep you informed as we investigate and address the issue. We ask that you give us reasonable time to resolve the vulnerability before any public disclosure.
